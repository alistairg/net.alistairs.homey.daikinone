If you have a Daikin One Plus, Daikin One Touch, or Skyport thermostat, this app connects it to Homey. You can control heating and cooling, turn the system on or off, adjust fan speed, and see the current status of each thermostat.

Setup is a one-time login with your Daikin account; the app talks to the official Daikin Skyport Cloud API, so no extra hardware or network configuration is required.

This is an unofficial integration and is not affiliated with or endorsed by Daikin Industries.
